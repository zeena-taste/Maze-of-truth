# Maze-of-truth
Mask of truth is a top-down puzzle game inspired by RPG style games. You navigate a maze and try to escape

Download game file from itch.io or github run the html code and run the game.html file

Game instruction:
- use WASD to move
- use M to toggle mask on and off which allows you to see extra path ways and the coins
- Collect all coins to be able to unlock the hidden door
- once you finish level 3 you win
